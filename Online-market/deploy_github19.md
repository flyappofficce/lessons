# Розміщення онлайн-магазину на GitHub Pages

У цьому файлі ми розглянемо, як покроково розмістити ваш проект онлайн-магазину на GitHub Pages, щоб він був доступний через Інтернет.

---

## Крок 1. Створення репозиторію на GitHub

1. Перейдіть на [github.com](https://github.com) і увійдіть у свій акаунт.
2. Натисніть **New repository** (Створити репозиторій).
3. Введіть назву репозиторію, наприклад: `online-shop`.
4. Додайте короткий опис (необов’язково).
5. Виберіть **Public**.
6. Не додавайте README, `.gitignore` або ліцензію зараз (ми додамо все локально).
7. Натисніть **Create repository**.

---

## Крок 2. Підготовка локального проекту

1. Відкрийте ваш проект в VS Code (або іншому редакторі).
2. Впевніться, що структура проекту така:

online-shop/
├── index.html
├── css/
│ └── style.css
├── js/
│ └── script.js
└── products/
├── tovar1.html
└── tovar2.html


3. Відкрийте `index.html` у браузері і перевірте, що все працює: сторінки, посилання, кнопки, стилі.

---

## Крок 2. Підготовка репозиторію на GitHub

Якщо репозиторій ще не створено:
1. Зайдіть на [GitHub](https://github.com) і увійдіть у свій акаунт.
2. Натисніть **New repository**.
3. Назва: `online-shop`.
4. Виберіть **Public** та натисніть **Create repository**.

Якщо репозиторій уже існує — переходимо до наступного кроку.

---

## Крок 3. Підключення локального проекту до GitHub

1. Відкрийте термінал у VS Code в кореневій папці проекту.
2. Виконайте команди:

```bash
git init                # якщо ще не ініціалізовано
git add .
git commit -m "Готовий проект онлайн-магазину"
git branch -M main
git remote add origin <URL_вашого_репозиторію>
git push -u origin main
```
<URL_вашого_репозиторію> — скопійований HTTPS або SSH URL вашого репозиторію.

## Крок 4. Активація GitHub Pages
1. Перейдіть у ваш репозиторій на GitHub.
2. Відкрийте вкладку Settings → Pages.
3. В секції Build and deployment:
- Source: Deploy from a branch
- Branch: main   (або master в залежності в якій гілці розміщено проект)
- Folder: / (root)
4. Натисніть Save.
5.  Через декілька хвилин GitHub згенерує посилання на ваш сайт, наприклад:
https://<ваш_нікнейм>.github.io/online-shop/

## Крок 5. Перевірка роботи сайту
1. Перейдіть за отриманим посиланням.
2. Переконайтесь, що:
- index.html відкривається
- всі сторінки товарів (products/tovar1.html, products/tovar2.html) працюють
- стилі та JS підключені
- всі hover-ефекти та інтерактивність працюють

## Крок 6. Оновлення проекту
Після змін у проекті:

```bash
git add .
git commit -m "Оновлення проекту"
git push

```
GitHub автоматично оновить сайт.
