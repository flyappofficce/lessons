# Урок 3: Списки та таблиці

## Крок 1. Категорії товарів

У цьому кроці ви навчитеся створювати списки категорій товарів.  
Для зручності ми зробимо **окрему сторінку для категорій**.

### Створення файлу сторінки категорій

1. У папці `products` створіть файл `categories.html`.
2. У файлі додайте заголовок та блок для списку:

```html
<h2>Категорії товарів</h2>
<!-- Список категорій -->
<ul>
  <li>Ноутбуки</li>
  <li>Смартфони</li>
  <li>Аксесуари</li>
</ul>
```

- Порада:
  -Ми використовуємо < ul > для категорій, бо порядок елементів не важливий.
  -Не потрібно писати весь HTML-документ — залишаємо тільки контент, який вставлятимемо в < body >.

## Крок 2. Додавання посилання на сторінку категорій у меню

Відкрийте index.html.
Додайте посилання у ваше меню:

```html
<!-- Меню навігації -->
<nav>
  <a href="products/categories.html">Категорії товарів</a>
</nav>
```
## Крок 3. Сторінка “Кроки покупки”

1. Створіть файл `purchase_steps.html` у папці `products`.
2. Додайте заголовок та нумерований список для кроків покупки:

```html
<h2>Як купити товар</h2>
<ol>
  <li>Виберіть товар у каталозі</li>
  <li>Натисніть "Детальніше"</li>
  <li>Додайте товар у кошик</li>
  <li>Оформіть замовлення</li>
</ol>
```
3. Додайте посилання на цю сторінку в меню index.html
```html
<nav>
  <a href="products/categories.html">Категорії товарів</a>
  <a href="products/purchase_steps.html">Як купити</a>
</nav>
```
## Крок 4. Таблиці характеристик товарів

1. Створіть файл `product_specs.html` у папці `products`.
2. Додайте заголовок та приклад таблиці:

```html
<h2>Характеристики товару</h2>
<table border="1">
  <tr>
    <th>Параметр</th>
    <th>Значення</th>
  </tr>
  <tr>
    <td>Процесор</td>
    <td>Intel Core i5</td>
  </tr>
  <tr>
    <td>Пам'ять</td>
    <td>8 GB</td>
  </tr>
</table>
```
3. Додайте посилання на цю сторінку в меню.

## Крок 6. Підсумок уроку

- Створили **окремі сторінки** для категорій, кроків покупки та таблиць характеристик.  
- Додали **посилання на ці сторінки в меню** на головній сторінці.  
- Навчилися робити сайт **структурованим та зрозумілим**, а не одним великим листком.  
- Ознайомилися з базовими тегами: `<ul>`, `<ol>`, `<table>`, `<th>`, `<td>`.

> Наступного разу переходимо до **Уроку 4: Форми та input-и**, де додамо форму для замовлення товару.
