# Урок 1: Вступ до HTML та підготовка проєкту "Онлайн магазин"

## Мета уроку
У цьому уроці ви навчитеся:
- Створювати репозиторій на GitHub.
- Підготувати структуру папок для онлайн-магазину.
- Створювати перший HTML-файл для головної сторінки магазину.
- Ознайомитися з базовими тегами HTML та семантикою сторінки.

---

## Крок 1. Створення репозиторію на GitHub

1. Відкрийте [GitHub](https://github.com/) та увійдіть у свій акаунт.
2. Натисніть кнопку **New repository**.
3. Введіть назву репозиторію: `online-store`.
4. В полі "Description" можна написати: `Проєкт онлайн магазину для курсу HTML/CSS/JS`.
5. Поставте галочку **Initialize this repository with a README**.
6. Натисніть **Create repository**.

Вітаю! Ваш репозиторій готовий.

---

## Крок 2. Клонування репозиторію 

- Можна відкрити репозиторій у браузері та натиснути **Code → Download ZIP**, розпакувати його на комп’ютері.
- Або, якщо ви знайомі з Git, можна скопіювати посилання для клонування і вставити його в GitHub Desktop або термінал.

---

## Крок 3. Підготовка робочої папки на комп’ютері

1. Відкрийте папку вашого проєкту (розпаковану з ZIP або клоновану).
2. Створіть всередині наступні папки:
   - **css** – для стилів сайту.
   - **js** – для скриптів JavaScript.
   - **images** – для картинок товарів та банерів.
   - **products** – для окремих сторінок товарів.
3. Перевірте, щоб у кореневій папці був файл `README.md`.

---

## Крок 4. Встановлення редактора коду

- Рекомендуємо [Visual Studio Code (VS Code)](https://code.visualstudio.com/) – безкоштовний редактор для роботи з HTML, CSS та JS.
- Інші редактори теж підходять (Sublime Text, Atom, Brackets).
- Відкрийте вашу папку проєкту у редакторі.

---

## Крок 5. Створення першого HTML-файлу

1. У кореневій папці створіть файл **index.html**.
2. Відкрийте його у вашому редакторі коду (VS Code або інший).
3. Напишіть базову структуру HTML-документа, використовуючи знання з цього уроку:
   - Почніть з декларації типу документа: `<!DOCTYPE html>`.
   - Додайте кореневий елемент `<html>` з атрибутом `lang="uk"`.
   - Створіть `<head>` з тегом `<title>` для назви сторінки.
   - Додайте `<meta charset="UTF-8">` для правильного відображення символів.
   - Створіть `<body>`, де розмістите контент сторінки.
4. Всередині `<body>` додайте:
   - **Заголовок магазину** за допомогою `<h1>` або `<h2>`.
   - **Короткий опис** сторінки або привітання у `<p>`.
5. Використовуйте коментарі `<!-- ... -->`, щоб пояснювати свої теги та блоки коду.  
   Наприклад:  
   `<!-- Тут буде назва магазину -->`.

---

## Крок 6. Перевірка роботи

1. Збережіть файл `index.html`.
2. Відкрийте його у браузері (два кліки на файлі).
3. Переконайтеся, що бачите:
   - Заголовок магазину.
   - Абзац з описом або привітанням.
   - Список категорій.
4. Якщо все відображається, ви успішно створили свій перший HTML-документ.

---

## Крок 7. Типові помилки на цьому етапі

- Забули декларацію `<!DOCTYPE html>`.
- Не закрили тег `<html>`, `<head>` або `<body>`.
- Неправильно закрили заголовки або абзаци.
- Не вказали `<meta charset="UTF-8">` у `<head>`.

Перевіряйте свій код уважно, використовуйте коментарі, і тоді сторінка завжди відобразиться правильно.

---

## Крок 8. Збереження у репозиторії

1. Відкрийте GitHub Desktop або інший Git-клієнт.
2. Додайте зміни до коміту.
3. Напишіть повідомлення коміту, наприклад:  
   `"Урок 1: створено перший HTML-файл з базовою структурою"`
4. Натисніть **Commit and Push**.

---

## Підсумок уроку

Ви:
- Написали перший HTML-документ з базовою структурою.
- Використали заголовки, абзаци та списки.
- Навчилися перевіряти сторінку у браузері.
- Вивчили типові помилки новачків.

> Наступний урок ми навчимося додавати **посилання та зображення товарів**, щоб сторінка почала виглядати як справжній каталог.
